🔧 One-time Firebase setup needed

NOJU uses Firebase to sync votes across devices in real-time (it's free!).
Open groupvote.html in a text editor and replace YOUR_API_KEY with your real Firebase config.

  1. Go to console.firebase.google.com
  2. Create a project → Add a web app → copy the config object
  3. Enable Realtime Database → Start in test mode
  4. Paste the values into the firebaseConfig block in this file
  5. Host on Netlify Drop or GitHub Pages for a public URL

💡 Pass-the-Phone mode works without Firebase — just pick "Vote on This Device" on the home screen.
